# Command Bookmarks 插件

dinotty 插件，用于收藏常用命令并批量发送到多个终端。支持多行命令、终端面板选择和快捷命令面板。

## 功能

- 添加、编辑、删除收藏命令（支持多行）
- 上下调整收藏顺序
- 从终端面板列表中选择目标终端（支持全选/取消全选）
- 一键将命令发送到所有选中的终端
- Command Palette 快速选择模式（`Command Bookmarks`），发送到当前活动终端
- 命令数据持久化存储

## 使用

### 主界面

左侧为收藏命令列表，右侧为目标终端选择面板。

```
┌─────────────────────────────────────────────────┐
│  Command Bookmarks                              │
├───────────────────────────┬─────────────────────┤
│  收藏命令           3 条   │  目标终端       全选  │
│  [+ 添加]                 │                     │
│                           │  ☑ Terminal 1       │
│  ↑ ↓ 编辑 删除 发送       │  ☐ Terminal 2       │
│  deploy-prod              │  ☑ Terminal 3       │
│  git pull && npm i        │                    │
│                           │                     │
│  ↑ ↓ 编辑 删除 发送       │                     │
│  docker compose up -d     │                     │
└───────────────────────────┴─────────────────────┘
```

### 添加命令

1. 点击 `+ 添加` 展开表单
2. 输入命令名称和命令内容（支持多行）
3. 点击 `保存` 或按 `Ctrl/Cmd+Enter` 快速保存

### 发送命令

1. 在右侧勾选目标终端
2. 点击命令右侧的 `发送` 按钮
3. 命令将同时发送到所有选中的终端

### 快速模式

通过 Command Palette 打开 `Command Bookmarks`，直接选择命令发送到当前活动终端。

## 目录结构

```
command-bookmarks/
├── README.md       # 本文档
├── plugin.json     # 插件清单
├── main.js         # 插件源码
└── styles.css      # 样式
```

## 安装

无需构建步骤，直接在 dinotty 插件管理器中安装，或手动链接：

```bash
ln -s $(pwd) ~/.dinotty/plugins/command-bookmarks
```
