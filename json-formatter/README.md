# JSON Formatter 插件

dinotty 插件，提供 JSON 格式化、压缩、验证和树形查看功能。

## 功能

- **格式化**：可配置缩进（2 空格 / 4 空格 / Tab）
- **压缩**：移除所有空白，输出最小 JSON
- **排序 Keys**：按字母顺序排列对象键
- **树形视图**：可折叠的 JSON 树，带语法高亮
- **Raw 视图**：纯文本格式化输出
- **错误提示**：JSON 解析错误详细显示
- **剪贴板操作**：一键粘贴输入、复制输出
- **回填**：将输出回填到输入，支持链式操作
- **统计信息**：输入/输出的字节数和行数

### 语法高亮

| 类型 | 颜色 |
|------|------|
| 键名 | 蓝色 |
| 字符串 | 浅蓝色 |
| 数字 | 金色 |
| 布尔值 | 紫色 |
| null | 灰色 |

### 树形视图操作

- 点击 `▶` / `▼` 展开/折叠节点
- `全折叠`：折叠所有节点
- `展开全部`：展开所有节点
- 折叠时显示子元素计数（如 `5 items` / `3 keys`）

## 使用

```
┌─────────────────────────────────────────────────────────┐
│  JSON Formatter         ☐ 排序 Keys  缩进: [2 ▾]       │
├─────────────────────┬───────────────────────────────────┤
│  输入         128B  │  输出    5 行 · 156B   [树][Raw]  │
│  [粘贴] [清空]      │  [全折叠] [展开全部] [复制]        │
│                     │                                   │
│  {                  │  {                                │
│    "name": "test"   │    "name": "test",               │
│  }                  │    "value": 42                    │
│                     │  }                                │
├─────────────────────┤                                   │
│  [格式化 →]         │                                   │
│  [压缩 →]           │                                   │
│  [← 回填]           │                                   │
└─────────────────────┴───────────────────────────────────┘
```

## 安装

无需构建步骤，直接在 dinotty 插件管理器中安装，或手动链接：

```bash
ln -s $(pwd) ~/.dinotty/plugins/json-formatter
```
