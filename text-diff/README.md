# Text Diff 插件

dinotty 插件，提供文本差异对比工具，支持逐行对比和字符级高亮。

## 功能

- 两栏输入（原始文本 vs 修改文本）
- LCS 算法计算差异
- **并排视图**：左右对照显示
- **统一视图**：单栏合并显示
- **字符级高亮**：相邻增删行自动标注差异字符
- 统计信息：新增/删除行数
- 交换和清空操作
- 响应式布局：窄屏自动上下堆叠
- 大文本保护：超过 50,000 行或 2,000,000 单元格时自动降级

## 使用

```
┌─────────────────────────────────────────────────────┐
│  Text Diff                                          │
├───────────────────────┬─────────────────────────────┤
│  原始文本             │  修改文本                    │
│                       │                             │
│  function hello() {   │  function hello(name) {     │
│    console.log('hi')  │    console.log(`hi ${name}`)│
│  }                    │    return true              │
│                       │  }                          │
│                       │                             │
├───────────────────────┴─────────────────────────────┤
│  [对比] [交换] [清空]  [并排]  +2 -1                │
├─────────────────────────────────────────────────────┤
│  1  function hello() {       │ 1  function hello(name) {
│  2    console.log('hi')      │ 2    console.log(`hi ${name}`)
│  3  }                        │ 3    return true
│                              │ 4  }
└──────────────────────────────┴──────────────────────┘
```

### 视图模式

- **并排**：左右两栏对照，新增行在右栏高亮，删除行在左栏高亮
- **统一**：单栏合并显示，`+` 前缀表示新增，`-` 前缀表示删除

### 字符级高亮

当相邻的删除行和新增行内容相似时，自动进行字符级差异标注：
- 删除的字符用红色背景标记
- 新增的字符用绿色背景标记

## 目录结构

```
text-diff/
├── README.md       # 本文档
├── plugin.json     # 插件清单
├── main.js         # 插件源码（含 LCS diff 算法实现）
└── styles.css      # 样式
```

## 安装

无需构建步骤，直接在 dinotty 插件管理器中安装，或手动链接：

```bash
ln -s $(pwd) ~/.dinotty/plugins/text-diff
```
